Introducing PCB111000 projects
	
This repository contains sample projects for PCB111000.  They are designed to illustrate the following features C.

	Simple C programs designed to drive a dual led or the display.
	The operation and uses of the basic logic gates.
	Text and numeric strings, and the conversion between numeric strings and decimal numbers.
	Binary and hex numbers.
	How numerical digits are made up of individual segments on the display.
	The acquisition of numbers from the keyboard leaving the mini-OS to to drive the display. 	
	The operation of simple interrupts.
	How to keep and display the time
	Input from the user switches
	How real numbers and scientific numbers can be reduced to simple forms 

Projects include
Random numbers
Prime numbers
Arithmetic operations
